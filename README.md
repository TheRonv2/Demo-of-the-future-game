# Demo

Этот репозиторий содержит демонстрацию уникальной концепции хоррор-игры, разработанной на Unity. Игра комбинирует 2D и 3D окружения для создания особого игрового опыта в жанре хоррор.

# Дизайн

Игра в основном разворачивается в подземной лаборатории, доступ к которой открывается к концу демонстрации. Изначально игроки исследуют ветхий дом, расположенный над лабораторией. Дом имеет заброшенный внешний вид, который не отражает его истинной природы.

По мере продвижения игроков окружающая среда претерпевает тревожные трансформации, демонстрируя уникальные архитектурные решения, которые подчеркивают игровые механики. Игра акцентирует внимание на атмосферном напряжении с минимальным использованием страшных моментов, за исключением случаев смерти персонажа, когда различные монстры игры мелькают на экране быстро подряд (похоже на "Soma").

Визуальное оформление игры в основном состоит из пиксельных изображений, найденных онлайн и модифицированных для достижения эстетики игры. Исключения включают в себя дизайн основного персонажа и определённые анимации, созданные вручную, несмотря на ограничения, связанные с относительным неопытом команды.

# Звуковой дизайн

Центральным элементом сюжета является жизненная борьба главного героя с навязчивыми шумами в его уме, подавленными медикаментами. При входе в дом эти шумы усиливаются, превращаясь в голоса, которые то помогают, то мешают герою на протяжении игры. При приближении опасности эти голоса нарастают, усиливая давление как на персонажа, так и на игрока.

# Геймплей

Основная цель демонстрации - показать разнообразные сегменты игрового процесса, включая:

Механику скрытности: перемещение от укрытия к укрытию, избегая источников света, которые могут обнаружить и повредить игрока, подобно "The Evil Within", адаптированному для 2D пространства.

Управление персонажем: внимательное управление движением персонажа, чтобы избежать опасных участков, например, спуск по трубе.

Избегание врагов: маневрирование вокруг монстров, чувствительных к свету, стратегическое включение фонарика для безопасного продвижения.

Демонстрация завершается последовательностью QTE (Quick Time Event), переводящей игроков в 3D сегмент игры.

Заключение
Этот проект нацелен на инновации в жанре хоррор, сочетая 2D и 3D окружения и придающий приоритет атмосферному напряжению перед традиционными страшными моментами. Демонстрация служит предварительным просмотром уникальных механик игры и элементов сюжета.
